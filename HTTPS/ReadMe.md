
Transport Layer 에 SSL 프로토콜위에
Application Layer의 HTTPS 프로토콜이 동작한다

보안이다.

SSL 보안 방식은 
-대칭키 암호방식(AES)
-공개키 (비대칭키) 암호방식(RSA)

대칭키 암호방식은 뚫리기 쉽고 빠르다.
공개키 암호방식은 뚫리기 어렵고 느리다.

HTTPS는 둘다 적절하게 사용한다
세션키 전송시 공개키 암호방식으로 전송
데이터 전송시 대칭키 암호방식으로 전송

이렇게 할시 뚫리기 어렵고 빠르게 전송가능하다.


RSA.html 코드는 위키에서 퍼옴.
